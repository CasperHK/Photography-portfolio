## Photo Gallery Page
When user click into the gallery photo, it redirect user into the gallery photo page. It show gallery photo in url like /gallery/1 . Left-hand side will show the gallery content informations including title, description, share button. 

## Creating Photo Detail Page & S
The photo shown in the home page horizontal grid will be linked to the photo detail page, url lilke /gallery/1/photo/3. 

## Share Dialog and Share Button
There will be a share button for user to open a share dialog with the following options circles:
1. Copy Link
2. WhatsApp
3. Facebook
4. Thread
5. X

You can make a BaseDialog for model reuse. And, create a ShareDialog for sharing.
